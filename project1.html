The purpose of this assignment is to get the students familiar with HTML and using basic HTML tags to create a static web site.


Requirements

  • Min of 3 HTML pages linked together
  • Usage of HTML 5 tags
  • Usage of <a> to jump to specific section
  • Usage of <ol> tag
  • Usage of <ul>
    1. Nested list
  • Usage of <table> tag
  • Usage of forms tags
  • Usage of <nav> and <footer> tags
  • Usage of <img>
  • Some kind of styling is being applied to website
  • User friendly
  • The website reflects some basic designs and thoughtfulness
  • No JavaScript or CDN libraries when working on this assignment

The content of the website could be anything the team decides on, as long as its appropriate. I want to open the content to the teams to allow for more team collaboration.
